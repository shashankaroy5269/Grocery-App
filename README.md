A modern Grocery Shopping App built with React and TanStack Query. It supports static data fetching, cart management, and smart discount & coupon system.

 Features
. Search, Filter & Sort items
. Add / Remove items from cart
. Fast data fetching with TanStack Query
. Automatic discount based on total
. Apply coupon codes
.Undo cart actions

 Tech Stack
.React (Next.js)
.TanStack Query
.CSS Modules
.Custom Hooks

 Discounts & Coupons
Auto discount on total amount
.Coupons: SAVE10, SAVE20, SUPER30, FLAT50

Run Locally
npm install
npm run dev
